# CodeAlpha_Credit_Scoring_Model
A Machine Learning model to predict creditworthiness using the Random Forest algorithm.
# Task 1: Credit Scoring Model 

## Project Overview
This project aims to predict an individual's creditworthiness using historical financial data. By applying machine learning classification algorithms, we can help financial institutions minimize risks by identifying potential defaulters.

## Dataset
- **Source:** UCI Machine Learning Repository (Statlog German Credit Data).
- **Features:** Includes 24 attributes such as financial history, income, loan duration, and employment status.
- **Target:** Binary classification (1: Good Credit, 0: Bad Credit).

## Technologies Used
- **Python 3.x**
- **Pandas** (Data Manipulation)
- **Scikit-learn** (Model Building)
- **Seaborn & Matplotlib** (Data Visualization)

## Model Performance
- **Algorithm:** Random Forest Classifier
- **Accuracy:** 81%
- **F1-Score:** 0.80 (Weighted Average)

## How to Run
1. Install dependencies: `pip install pandas scikit-learn matplotlib seaborn`
2. Run the script: `python credit_scoring.py`
